## Utilitario

- **Debloat Android**

  Remove aplicativos inuteis (Bloatware) utilizando o `adb`.

- **Download de Mp3**

  Download de Mp3 utilizando a blibioteca `yt-dlp`.

- **Gravação de Tela**

  Gravação de tela utilizando `ffmpeg`.

- **Backup**

  Backup local utilizando a ferramenta `rsync`.

## Requisitos

- Linux
- `adb`
- `ffmpeg`
- `yt-dlp`
- `rsync`

## ▶️ Uso

```bash
./script.sh
```

## ⚠️ Aviso

A remoção de aplicativos do sistema pode causar instabilidade.
Use por sua conta e risco !.
